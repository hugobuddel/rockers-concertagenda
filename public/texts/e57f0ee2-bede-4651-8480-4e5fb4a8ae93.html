Klaar met die standaard clubavonden en toe aan het tegenovergestelde? Of ben je niet zo van de hippe discos maar heb je toch zin in goed (dans)vermaak? Neushoorn biedt je een clubavond met alle toppers uit het alternatieve indiecircuit! Ga met ons terug naar die zaterdagnacht op Lowlands, waar je in de Grolsch-tent je benen lam springt bij de hipste indieklassiekers en meezingt met de lekkerste rockhits! Trek je hipste shirt maar aan en rock met ons mee 'When The Sun Goes Down!'. Denk aan: The Killers / Beastie Boys / QOTSA / Offspring / Arctic Monkeys / Green Day / Chemical Brothers / The Prodigy / The Black Keys / Manu Chao / The Smashing Pumpkins / Tame Impala / Guano Apes / Deftones / Twenty One Pilots / Underworld / Limp Bizkit / Placebo / Muse / System of a Down / Korn / Foo Fighters / Arcade Fire / Franz Ferdinand / Billie Eilish / Jake Bug / Blur / The White Stripes / Greta van Fleet / Faithless / Bloodhound Gang / Editors / DJVT / Kaiser Chiefs / Chvrches / Wolfmother / Fatboy Slim / The 1975 / Fun Lovin Criminals / Blood Red Shoes / JXL / The Hives / MGMT / My Chemical Romance / The Wombats / The Ting Tings / War On Drugs / Volbeat / Basement Jaxx / Eagles of Death Metal / Enter Shikari / Nirvana + vele anderen. Deze lijst is er om een indruk te geven wat je ongeveer kunt verwachten. Wees niet verbaasd als de dj's hiervan afwijken. Zaal: AA-Security Arena | Leeftijd: 16+ | Garderobe gratis
